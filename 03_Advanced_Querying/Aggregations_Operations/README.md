# SQL Aggregations Operations

## Overview
Aggregation functions summarize data: COUNT, SUM, AVG, MAX, MIN. Used with GROUP BY and HAVING.

The script `SQL_Aggregations_Operations.sql` covers functions, grouping, filtering aggregates in MySQL.

## Key Functions

### COUNT()
- Counts rows or non-NULL values.

### SUM()
- Totals numeric column.

### AVG()
- Average value.

### MAX/MIN()
- Highest/lowest value.

### GROUP BY
- Groups rows for per-group aggregates.

### HAVING
- Filters groups (WHERE for aggregates).

## Real-World Use Cases
- **Sales Reports**: Total revenue per category.
- **Analytics**: Average order value.
- **Dashboards**: Top products by sales.
- **Inventory**: Count low-stock items.

## Best Practices
- Pair aggregates with GROUP BY.
- Use HAVING for group conditions.
- ROUND() for clean averages.
